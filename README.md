# 📊 Proyecto Módulo 1 – Fundamentos de Programación

---

## 📝 Descripción

Aplicación desarrollada en **Streamlit** que integra conceptos fundamentales de programación como variables, condicionales, estructuras de datos, funciones, programación funcional y Programación Orientada a Objetos (POO).

Este proyecto corresponde al primer trabajo aplicado del curso y forma parte de la construcción de un portafolio personal.

---

## 🚀 Funcionalidades Implementadas

- 💰 Verificador de presupuesto (variables y condicionales)
- 📋 Registro de actividades financieras (listas y diccionarios)
- 📈 Cálculo de retorno esperado usando programación funcional (map y lambda)
- 🏗 Modelado de actividades mediante Programación Orientada a Objetos (POO)

---

## ▶️ Instrucciones de Ejecución

1. Clonar el repositorio:
   git clone (colocar enlace del repositorio)

2. Instalar dependencias:
   pip install -r requirements.txt

3. Ejecutar la aplicación:
   streamlit run app.py
